# 👕 Fashion Drip Website

A modern streetwear website built for **Fashion Drip**, a local clothing brand based in **Kurukshetra**.

## About the Project

This project was developed to showcase my frontend development skills, UI design sense, and ability to build modern, responsive websites for real-world businesses.

## Features

* Modern dark-themed UI
* Fully responsive design
* Smooth animations and transitions
* Product showcase section
* Interactive elements using JavaScript

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Note

This project was initially created with the help of AI tools and then customized, structured, and enhanced by me.
